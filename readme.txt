A minimal, fast landing page WordPress theme with hero, features, testimonials, pricing, and contact form.

== Description ==

Dental Care is a lightweight, responsive WordPress theme designed for one-page sites such as dental clinics, small businesses, and landing pages. It includes sections for hero content, features, testimonials, pricing tables, and a contact form. The theme fully supports:

* Custom logo
* Featured images (post thumbnails)
* Navigation menus
* Block editor styles and patterns
* Responsive embeds
* Post tags and author avatar display
* Widget-ready sidebar
* Custom CSS via theme options

Optional support for **Advanced Custom Fields (ACF)** to manage custom hero subtitles and other fields.

== Installation ==

1. Upload the theme zip via Appearance → Themes → Add New → Upload Theme.
2. Activate the theme.
3. Create a page and set it as the **Front Page** in Settings → Reading.
4. Use the page title as the hero title, and either the excerpt or ACF field `mlt_subtitle` as the hero subtitle.
5. (Optional) Install **Advanced Custom Fields** plugin to manage extra fields.

== Frequently Asked Questions ==

= Can I use this theme for other small business landing pages? =

Yes! The theme is fully customizable and can be adapted for other landing pages beyond dental clinics.

= Does this theme support Gutenberg blocks? =

Yes! The theme supports Gutenberg blocks, block patterns, wide/alignment options, and responsive embeds.

= How do I add a sidebar or widgets? =

The theme includes a registered sidebar that you can manage via Appearance → Widgets.
